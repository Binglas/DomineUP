Criar Sala de Jogo

Para criar uma sala de jogo o utilizador deve selecionar o botão "Create Room" no Welcome Screen:

Este botão dá acesso a uma nova janela onde se deve escolher o nome da sala, o número máximo de jogadores e a password (facultativa).

O nome da sala de jogo não pode ser igual a nenhuma outra que esteja online de momento senão aparecerá a seguinte mensagem de erro:

Em caso de sucesso na criação da sala, esta aparecerá na lista de salas online do Welcome Screen.
